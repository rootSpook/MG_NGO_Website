Help

How can we help you?

search

article Creating & Managing Blog Posts

Go to Blog Posts in the sidebar to view all posts. Click New Post to create one. Fill in the title, content, category and status, then click Publish or Save as Draft. You can edit or delete any post from the list using the action buttons.

event Adding & Managing Events

Navigate to Events to see all scheduled events. Use Add Event to create a new one. Provide a title, date, time, location, description and cover image. Events can be saved as drafts before publishing.

calendar_month Using the Calendar

The Calendar page shows a full year view with working and non-working day counts. Use the arrows to navigate between years. Click Add Event in the top-right corner to schedule a new event. Weekends are highlighted in red.

perm_media Uploading Media

Go to Media and click the upload area or drag & drop files to upload images, videos or PDFs. Uploaded files appear in the Media Library below. Hover over any item to reveal the delete button.

campaign Publishing Announcements

Use the Announcement page to post important notices to the website. Click New Announcement, fill in the title and message, set the status to Active and save. Drafts are not visible to visitors until set to Active.

contact_support Contact Support

If you encounter any issues, please contact the site administrator at admin@example.com. Include a brief description of the problem and any error messages you see.
